Radio Thermostat
================

The **RadioThermostat** Smart Device allows you to integrate
[Filtrete 3M-50](http://www.radiothermostat.com/filtrete/products/3M-50/)
WiFi thermostat into the [SmartThings](http://fbuy.me/bb9pe) home automation
system.
